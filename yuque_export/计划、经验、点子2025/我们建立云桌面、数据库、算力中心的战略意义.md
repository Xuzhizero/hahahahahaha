1. 拥有对基础设施的了解，无可替代



云桌面：

一是**提升资源利用率**，比如一台物理服务器上跑多个云桌面；  
二是**集中管理、统一配置，比如软件环境统一（一个界面完成90%运维工作）、权限管控、运维（系统安装、升级、软件更新，一个运维人员管理1000个终端小case）方便**；  
三是**增强数据安全性，**比如本地不落地、统一备份。

四是省电（深信服给的数据是70%）

五是如果软件损坏，可以迅速恢复



成熟的云桌面当然有它的好。就像数据库和算力中心一样。但是如果用起来非常硌得慌，还不如不用。所以关键在于体验。



深信服的优势：

个人盘加密技术：运维人员也无法查看个人隐私

传统桌面云需要大量组件，深信服让桌面云变得简洁，同时提供VMS（虚拟服务器软件），VDC（虚拟桌面控制器）、瘦终端三大组件，提供一站式桌面云方案

> <font style="color:rgba(0, 0, 0, 0.85);">VMS 是一类用于在物理服务器上创建、管理和运行多个虚拟服务器（虚拟机，VM）的软件，其核心功能是通过</font>**<font style="color:rgb(0, 0, 0) !important;">虚拟化技术</font>**<font style="color:rgba(0, 0, 0, 0.85);">实现硬件资源（如 CPU、内存、存储、网络）的抽象与共享，让一台物理服务器可同时运行多个独立的操作系统（如 Windows、Linux 等）及应用程序，彼此隔离且互不干扰。</font>
>
> <font style="color:rgba(0, 0, 0, 0.85);"></font>
>
> <font style="color:rgba(0, 0, 0, 0.85) !important;">VDC 是虚拟桌面基础设施（VDI，Virtual Desktop Infrastructure）中的核心组件，负责</font>**<font style="color:rgb(0, 0, 0) !important;">管理和控制虚拟桌面的生命周期、用户连接及资源分配</font>**<font style="color:rgba(0, 0, 0, 0.85) !important;">，是用户设备与后端虚拟桌面之间的 “桥梁”。</font>
