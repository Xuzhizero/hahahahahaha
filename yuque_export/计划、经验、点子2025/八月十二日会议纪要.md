# 内部会议摘要
1. 代码评审以提升鲁棒性。
2. 加速仿真实验室建设。

# 槽道舟试验报告摘要
+ 高速测试基于雷达的避碰，涵盖追越、交叉、相遇三种情景；追越效果不佳，其余良好。
+ 精确雷达支持有效避碰算法。
+ 需要对雷达航向变化趋势进行滤波处理。
+ 建议由专业人士测试雷达参数。
+ 目标捕捉存在滞后问题。

## 遥控通信与控制问题
+ 寻求提高通信稳定性和距离的方法。
+ 当前加速响应过于激烈，导致过冲现象。

## 视频跟踪识别现状
+ 尝试了纯视觉和纯雷达两种方案，但都面临目标丢失的问题。
+ 当前识别概率为60%。

## 设备改进需求
+ 提升续航能力至至少8小时。
+ 设备需便于安装、维护，并具备长期耐用性和防水性能。

## 未来开发方向
+ 自动生成远离岸边（如180米外）的安全导航线。

## 雷达使用及优化建议
+ 现有雷达旋转频率无法满足高机动需求。
+ 考虑将AIS信息与雷达数据融合来改善避碰效果。
+ 对于AIS是否应为主要避碰手段的观点持保留态度。
