# 船舶远程控制中心试验大纲
## 1. 适用范围
本大纲适用于杭州陆地控制中心对舟山海域船舶远程控制系统的检测。

## 2. 检测依据
《船舶远程控制中心设计方案报告》、《船舶自动化系统技术规范》、《海上无线通信技术标准》。

## 3. 性能指标
+ 网络通信延时：≤150ms
+ 控制指令执行时间：普通指令≤200ms，紧急指令≤150ms
+ 无人机图像传输延时：1080P模式≤250ms，4K模式≤400ms
+ 船体状态数据更新频率：≥1Hz
+ 系统可用性：≥99.9%

## 4. 试验条件
1. 试验天气情况良好，海况不超过6级；
2. 试验区域在舟山海域开阔水域中，距离岸基控制中心不少于50海里；
3. 杭州陆地控制中心各设备运行正常；
4. 船载设备系统完整安装并调试完毕；
5. 通信链路（卫星通信+4G/5G移动通信）正常。

## 5. 试验内容
1. **网络通信性能试验**：
    - 延时测试：连续24小时不间断ping测试，试验次数≥3次；
    - 链路切换测试：主备链路故障切换，试验次数≥5次；
    - 数据吞吐量测试：不同网络负载下的传输性能，试验次数≥3次。
2. **控制指令下发试验**：
    - 航向控制指令：左右舵角调节，试验次数≥5次；
    - 速度控制指令：加速、减速、停车指令，试验次数≥5次；
    - 设备控制指令：发动机启停、辅助设备开关，试验次数≥3次；
    - 紧急控制指令：紧急停车、紧急转向，试验次数≥3次。
3. **无人机图像回传试验**：
    - 不同分辨率图像传输：1080P、4K模式，试验次数≥3次；
    - 不同天气条件测试：晴天、阴天、小雨，试验次数≥3次；
    - 连续飞行测试：单次飞行时间≥30分钟，试验次数≥3次。
4. **船体实时状态回传试验**：
    - GPS位置信息回传：精度测试，试验次数≥10次；
    - 航向信息回传：陀螺罗盘数据，试验次数≥10次；
    - 发动机参数回传：转速、油耗等，试验次数≥5次；
    - 传感器数据完整性：数据包完整率测试，试验次数≥3次。
5. **系统集成测试**：
    - 多功能并行操作：同时进行控制、监控、图像传输，试验次数≥3次；
    - 故障恢复测试：模拟设备故障后的系统恢复，试验次数≥3次。

## 7. 试验结论
通过本次试验验证，船舶远程控制中心系统各项性能指标均达到设计要求：

1. **网络通信性能**：在各种海况条件下，系统延时均控制在150ms以内，丢包率低于0.15%，链路切换功能正常。
2. **控制指令执行**：各类控制指令响应及时，执行精度满足要求，紧急控制指令优先级机制有效。
3. **无人机图像回传**：不同分辨率模式下图像传输稳定，延时符合预期，适应不同天气条件。
4. **船体状态回传**：传感器数据采集准确，传输完整率高，实时性良好。

系统具备投入实际运行的条件，建议进入下一阶段的长期稳定性测试。
