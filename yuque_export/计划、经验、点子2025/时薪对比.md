# 提问
<font style="color:rgb(31, 35, 41);">我们算一下时薪。 A：一天工作6.5h，一周5天。 一个月1.7万税前的月薪 B：一天工作9h，一周六天，一个月2.5万税前的月薪 对比A和B的税后时薪哪个更高</font>

<font style="color:rgb(31, 35, 41);"></font>

# <font style="color:rgb(31, 35, 41);">回答</font>
要对比A和B的税后时薪，需先计算各自的税前时薪，再考虑税后收入。由于中国个税计算涉及起征点、专项扣除等因素，以下按常规情况估算（假设无专项附加扣除，仅考虑基本减除费用5000元/月）：  



### **一、计算税前工作时长**
#### **A的工作时长**
+ 日工作时长：6.5小时  
+ 周工作天数：5天  
+ 月工作周数：按常规每月约4周计算（实际可能有差异，此处简化）  
+ **月总工作时长**：6.5小时/天 × 5天/周 × 4周 = **130小时**

#### **B的工作时长**
+ 日工作时长：9小时  
+ 周工作天数：6天  
+ 月工作周数：4周  
+ **月总工作时长**：9小时/天 × 6天/周 × 4周 = **216小时**



### **二、计算税前时薪**
+ **A的税前时薪**：17000元/月 ÷ 130小时 ≈ **130.77元/小时**  
+ **B的税前时薪**：25000元/月 ÷ 216小时 ≈ **115.74元/小时**



### **三、计算税后收入（估算）**
#### **1. 个税计算逻辑**
根据中国2025年个税政策（假设未调整），应纳税所得额 = 月薪 - 5000元（起征点） - 专项扣除（此处暂不考虑），税率按超额累进税率计算：  



#### **2. A的税后收入计算**
+ 应纳税所得额：17000 - 5000 = **12000元**  
+ 对应税率：10%（超过3000元至12000元部分）  
+ 速算扣除数：210元  
+ **每月个税**：12000 × 10% - 210 = 990元  
+ **税后月薪**：17000 - 990 = **16010元**  
+ **税后时薪**：16010元 ÷ 130小时 ≈ **123.15元/小时**



#### **3. B的税后收入计算**
+ 应纳税所得额：25000 - 5000 = **20000元**  
+ 对应税率：20%（超过12000元至25000元部分）  
+ 速算扣除数：1410元  
+ **每月个税**：20000 × 20% - 1410 = 2590元  
+ **税后月薪**：25000 - 2590 = **22410元**  
+ **税后时薪**：22410元 ÷ 216小时 ≈ **103.75元/小时**



### **四、对比结果**
| 项目 | A（1.7万/月） | B（2.5万/月） |
| --- | --- | --- |
| 税前时薪 | 130.77元/小时 | 115.74元/小时 |
| **税后时薪** | **123.15元/小时** | **103.75元/小时** |


### **结论**
**A的税后时薪更高**。尽管B的月薪更高，但由于工作时长更长，且个税税率更高，税后时薪反而低于A。
