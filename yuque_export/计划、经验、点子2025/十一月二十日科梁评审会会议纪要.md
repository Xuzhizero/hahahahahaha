# ✅ **九、联调过程中的问题与优化说明（重点）**
他们提到的所有问题都可以成为你的**提问点**。

### 1）发电机电压“无惯性”问题
**本质：模型一开始等效成“理想电压源”**  
→ 根本不符合发电机物理特性  
→ 通过加入惯性才修复

这是一个非常大的模型结构性问题。

### 2）发电机故障后 PI 参数异常漂移
+ 故障注入后 PI 发散
+ 无法自动恢复
+ 需人工加“使能”逻辑屏蔽

### 3）主发电机出现逆电流
+ 归因于整流桩侧阻尼电阻建模
+ 通过改位置解决

### 4）逆变器输出电压偏差
+ 380/220 偏离严重
+ 通过优化逆变器控制 + 滤波参数修复

### 5）左右推进器功率偏差
+ 归因于推进电机发热模型
+ 属正常现象（按他们的说法）

### 6）功率匹配单元模型发散
+ 原因来自“遥控模式电平抖动”
+ 乙方加延迟与异常点屏蔽模块修复

你们现场也讨论到**可能是单芯信号线无屏蔽导致耦合噪声**。

### 7）自动负荷分配异常
+ 发电机突然启动/信号突变
+ 乙方通过延迟滤波器修复
+ 控制器是否真的发送“异常指令”仍待验证
