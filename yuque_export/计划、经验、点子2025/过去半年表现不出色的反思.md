### **一、信任透支的现状**
+ 作者意识到自己已经透支了领导对自己的信任

+ 作者选择自己承担责任，但这些苦没有人知道

### **三、能力边界的认知**
+ 认识到自己需要清晰了解自己的能力边界
+ 决定坚决拒绝超出能力边界但很诱人的挑战
+ 强调要"求稳"
+ 当负责人需要足够的贡献和影响力，需要能够服众

### **四、当前的职业策略**
+ **不再相信任何领导对自己的看法**（无论看高还是看低）
+ 决定勤勤恳恳做好本职工作
+ 暂时不当负责人，但仍要"仰望星空"
+ 专注做好自己的工作，不带别人

### **五、未来是否担任负责人的考量维度**
评估需要从四个维度考虑：

1. **影响力**：是否有足够贡献让人信服
2. **能力**：技术能力是否足够
3. **经验**：特别是表达能力、判断能力、临场应变能力
4. **信念**：是否有源源不断的精神动力来源

### **六、当负责人的利弊分析**
**利**：有很强的自主性

**弊**：

+ 会占用大量精力在管理和文件工作上
+ 26岁就搞管理似乎不太合适
+ 如果没有足够积累就当负责人会很痛苦且结果不理想
+ 负责人需要前提条件，"没有金刚钻不揽瓷器活"

+ 赵工有经验但能力较弱，仍不能把团队带到安全位置

### **八、最终决定**
+ **决定不当负责人，只当普通技术员工**
+ 这样可以确保把技术工作做好，不用加班
+ 下班后可以把时间投入到为未来做准备
